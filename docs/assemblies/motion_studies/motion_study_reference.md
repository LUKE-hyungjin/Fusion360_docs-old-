# 모션 스터디 참조
접합을 기반으로 운동학 동작 분석을 수행합니다.

## 접합
접합의 이름을 표시합니다.

## 자유도(DOF)
자유도 유형을 표시합니다.

## 값
슬라이더가 있는 위치의 DOF 값을 표시합니다.

## 재생 버튼
재생 버튼을 사용하여 연속된 동작이나 단계별로 뒤로 및 앞으로 재생합니다.

## 모드
동작을 한 방향으로 앞으로 재생한 다음 뒤로 또는 루프로 재생합니다.

## 속도
슬라이더를 사용하여 재생 속도를 제어합니다.
